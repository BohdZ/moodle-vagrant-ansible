# moodle-vagrant-ansible
# Moodle Infrastructure with Vagrant + Ansible

This project deploys a full Moodle environment using:

- Vagrant (VM provisioning)
- Ansible (automation)
- Ubuntu 22.04

## Architecture

- Web VM: Apache + PHP + Moodle
- DB VM: MariaDB

## Requirements

- VirtualBox
- Vagrant
- Ansible

## Run project

```bash
vagrant up
